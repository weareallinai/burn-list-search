# 🔥 Burn List

**The song tracker for Motorcycle Tony's Music Club.**

## About

Burn List is where the crew logs the tracks they're burning — the songs stuck on repeat, the ones you can't stop playing. Search the list, add your own picks, and see album art pulled straight from Spotify.

## Features

- 🔍 **Search** songs by title or artist
- 🎨 **Album art** auto-fetched via the Spotify API
- 🔒 **Passcode-protected** song submissions
- 🖼️ **Batch backfill** for missing artwork
- 🌑 **Dark, fire-themed UI**

## Tech Stack

- [Vite](https://vitejs.dev) + [React](https://react.dev) + [TypeScript](https://www.typescriptlang.org)
- [Tailwind CSS](https://tailwindcss.com) + [shadcn/ui](https://ui.shadcn.com)
- Lovable Cloud (backend + database)
